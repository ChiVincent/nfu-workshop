# NFU Workshop

The demo for nfu laravel deploy workshop.

## Requirement

- PHP 7.2
- MariaDB
- Redis
- Yarn

## Usage

### Development

```
git clone https://github.com/chivincent/nfu-workshop.git
cd nfu-workshop

composer install 
yarn install 
yarn run dev 

php artisan migrate
php artisan serve
```
